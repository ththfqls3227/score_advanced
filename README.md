# 다트 심화 과제

## 학생 성적 분석 프로그램
텍스트 파일을 읽어와 학생 정보를 처리하는 Dart 콘솔 프로그램입니다.

### 1. Score / StudentScore 클래스 구성

Score 클래스에서 점수(mark)를 저장하고 출력하는 기능을 제공합니다.

StudentScore 클래스는 Score를 상속받아 이름(name)과 점수를 함께 관리합니다.

showInfo() 메서드를 재정의하여 "이름: 홍길동, 점수: 90점" 형식으로 출력합니다.

### 2. 학생 데이터 파일 읽기 기능

프로그램 시작 시 students.txt 파일을 읽어옵니다.

각 줄은 "이름,점수" 형식으로 되어 있으며, 쉼표 기준으로 분리하여 StudentScore 객체로 변환합니다.

잘못된 형식의 데이터가 있을 경우 예외 처리 후 프로그램을 종료합니다.

불러온 데이터는 List<StudentScore> 형태로 저장됩니다.

### 3. 사용자 입력 기반 학생 점수 조회 기능

사용자에게 어떤 학생의 점수를 확인할지 번호로 입력받습니다.

Windows 한글 입력 인코딩 문제로 인해 숫자 선택 방식으로 구현되었습니다.

1 입력 → 홍길동

2 입력 → 김철수

잘못된 번호를 입력하면 다시 입력하도록 안내합니다.

선택된 이름과 일치하는 학생들의 점수를 찾아 평균을 계산합니다.

### 4. 결과 저장 기능

평균 계산이 완료되면 "이름: 홍길동, 점수: 90점" 형식의 문자열을 생성합니다.

해당 문자열을 result.txt 파일에 저장합니다.

저장 완료 시 "저장이 완료되었습니다." 메시지를 출력합니다.
